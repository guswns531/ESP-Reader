# 한국어 노트

ESP-Reader는 이제 한국어 UI 폰트를 사용할 수 있습니다.

이 문서는 본문 줄바꿈이 한국어에서도 어느 정도 자연스럽게 동작하는지 확인하기 위한 샘플입니다.

## 읽기 목표

- 긴 문장이 화면 폭에 맞게 적절히 나뉘는지 본다.
- 제목과 본문이 구분되어 보이는지 확인한다.
- 목록과 일반 문단의 간격이 충분한지 체크한다.

## 테스트 문장

전자종이 화면에서는 빠른 애니메이션보다 안정적인 레이아웃과 선명한 텍스트가 더 중요하다.

사용자는 페이지를 넘길 때마다 현재 위치를 쉽게 인지할 수 있어야 하고, 선택된 항목도 한눈에 보이는 편이 좋다.
